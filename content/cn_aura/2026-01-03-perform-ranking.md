{{% details-summary summary="3. 已知总预算 1600 各个商品单价为300 500 700 如果至少买一件 采购总价大于1000的买法占所有买法的几分之几？" default="close" %}}
大意是历史上是百鸡问题，后续演变成运筹问题，动态规划等方法可以计算机求解他们。参考代码如下:

  ```
  def solve_with_dp():
    budget = 1600
    prices = [300, 500, 700]
    
    # dp[i] 表示花费刚好为 i 元的买法数量
    # 预算是 1600，所以数组长度 1601
    dp = [0] * (budget + 1)
    dp[0] = 1  # 基础：花费 0 元的方法只有 1 种（什么都不买）

    # 核心状态转移
    for p in prices:
        for j in range(p, budget + 1):
            dp[j] += dp[j - p]

    # 计算总数（排除花费 0 的情况）
    total_valid_schemes = sum(dp[1:budget + 1])
    
    # 计算大于 1000 的总数
    greater_than_1000 = sum(dp[1001:budget + 1])

    print(f"所有买法总数: {total_valid_schemes}")
    print(f"大于1000的买法数: {greater_than_1000}")
    print(f"占比: {greater_than_1000}/{total_valid_schemes}")

solve_with_dp()
  ```
  可以在https://www.online-python.com/运行上面代码，但是人类在草稿纸上思考方式并不推荐这种。推荐从大到小的方案遍历。

{{% details-summary summary="4.一笔画相关问题" default="close" %}}
你可以遵循以下三步来判断：

判断连通：图形必须是连通的（各部分有边相连）。

数奇点个数：从一个点出发，数清与它相连的线段数目。如果线段数是奇数，这个点就是奇点；是偶数则是偶点。

0个奇点：可以一笔画成，并且起点和终点为同一点（可以回到起点）。

恰好2个奇点：可以一笔画成，但必须从一个奇点出发，到另一个奇点结束 。

判断两笔画和三笔画等问题，可以转化为一个更通用的模型，核心结论是：

一个连通图至少需要的笔画数，等于它的奇点数目除以2。
公式：最少笔画数 = 奇点总数 ÷ 2

一、草稿本卷面整洁的“利”

思路清晰，逻辑有序
整洁的草稿本通常意味着使用者有意识地分区、分步骤书写，这本身就是对思路的实时整理。当需要回溯检查时，能快速定位到关键步骤，便于复查和纠错。尤其是在解决复杂数学题或推导时，清晰的逻辑链条一目了然。
提升专注度，减少干扰
整洁的卷面可以减少视觉噪音，让使用者专注于当前的计算或思考，不被杂乱的笔迹分散注意力。这种秩序感能营造更高效的思维环境。
培养严谨的思维习惯与规范意识
长期保持草稿整洁，有助于培养系统性、条理化的思维习惯，潜移默化地提升做事和解决问题的结构化能力。在考试中，这种习惯也能自然迁移到答卷上，减少因步骤混乱导致的失分。
便于复习与整理错题
整洁的草稿可以作为学习过程的有效记录。复习时，可以回顾当时的解题思路，特别是对于难题，草稿本上的思考痕迹是宝贵的“思维档案”。整理错题时，也能直接引用清晰的推导过程。
心理上的积极暗示
一个整洁的环境（包括草稿本）往往能带来更强的掌控感和自信心，减少因杂乱而产生的焦虑或烦躁情绪。

二、草稿本卷面整洁的“弊”

可能拖慢思维速度，打断连续性
追求整洁有时需要付出额外的时间成本：画线分区、擦除修改、刻意保持工整等。对于需要高速运转、跳跃性思维的场景（如限时考试、头脑风暴），过分关注形式可能抑制思维的流畅性和 spontaneity（自发性），打断思考的连贯性。
可能产生“形式主义”压力
对整洁的过度追求可能演变为一种心理负担，尤其是对完美主义者而言。他们可能因为担心弄乱页面而不敢大胆尝试、涂画，反而束缚了探索性思维。草稿的本质是“草”，是思维的试验田，不应成为另一个需要精心维护的作品。
可能掩盖真正的思考过程
真正的思考往往是非线性、试错式、充满涂改和标记的。过度整洁的草稿有时会擦除这些“错误”或“旁支”路径，而这些痕迹恰恰是理解思维漏洞、发现新思路的关键。杂乱的草稿可能更真实地反映认知过程。
不适合所有思维类型
许多创造性人才、科学家、工程师的草稿往往看似混乱，实则内嵌着他们独特的思维地图（如关联线、关键词、图形）。强制整理可能破坏他们特有的信息组织方式，降低效率。
三、核心辩证与建议

草稿本的终极目的是 服务于高效思考与问题解决，而非追求形式美观本身。因此，其卷面是否整洁应取决于 使用场景、个人思维习惯与当前任务性质。

区分场景：

日常练习与复习：建议有意识地适度整洁，以培养条理性和便于复查。
考试与限时任务：应以速度为优先，在保证自己能看懂关键步骤的前提下，无需苛求工整。
探索性思考与创意阶段：应彻底放开，鼓励随意涂画、建立连接，哪怕纸面“一片狼藉”。
个人化策略：

可以尝试“结构化杂乱”的折中方法：不追求字迹工整，但用箭头、圈画、分区线等简单标记保持基本秩序，在自由与有序间找到平衡。
了解自己是“线性思维者”（适合整洁有序）还是“视觉/发散思维者”（适合图形化、关联式草稿），选择最适合自己的方式。
结论

草稿本卷面整洁与否，本质上是 思维效率与形式规范之间的权衡。
利在于促进条理、便于复查、培养规范；弊在于可能抑制思维速度、增加心理负担、掩盖真实过程。

最高原则是“功能性至上”：让草稿本成为你思维最得力的助手，而不是负担。理想的草稿，应是思考过程最真实、最有效的延伸——无论是井然有序，还是看似混乱却自成体系。找到那个能让你思维最流畅、问题解决最有效的状态，便是最好的“整洁”。

<span style="color: red;">关键在于实用性，可以用于后续复盘优化，而不是无从下手</span>
